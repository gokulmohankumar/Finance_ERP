# 🏢 Finance ERP

A role-based Enterprise Resource Planning (ERP) application developed to streamline business operations such as employee management, inventory tracking, and financial record management.

The project follows a modern full-stack architecture with a **Spring Boot** backend exposing RESTful APIs and a **React** frontend delivering a responsive user experience.

---

## ✨ Features

- 👤 Role-based authentication and authorization
- 👥 Employee management
- 📦 Inventory management
- 💰 Financial record management
- 🔄 RESTful API integration
- 📱 Responsive user interface
